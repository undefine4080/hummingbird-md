# Hummingbird MD

蜂鸟MD预览——一款简单、有用的 VS Code Markdown 阅读器插件

![截图](/media/preview.png)

## 功能特性

- 代码高亮：基于 [Shiki](https://shiki.style/) 的精确语法高亮，使用 VS Code 同款 TextMate 语法引擎，支持数百种编程语言，亮暗双主题自动切换。
- 目录导航：TOC 侧边栏生成文档目录树，点击标题快速跳转，阅读时实时高亮当前位置。
- Mermaid 图表：支持 Mermaid 语法渲染流程图、时序图、甘特图等，点击可全屏查看并导出为图片。
- 图片查看器：点击图片进入全屏查看模式，支持缩放、旋转、拖拽和前后切换。
- 主题切换：跟随 VS Code 主题自动适配亮色/暗色模式，也可在侧边栏手动切换。

## 使用方式

有三种方式打开预览：

1. **编辑器工具栏** — 打开 `.md` 文件后，点击编辑器右上角的蜂鸟图标
2. **文件资源管理器** — 右键点击 `.md` 文件，选择「Preview in Hummingbird MD」
3. **命令面板** — `Ctrl/Cmd + Shift + P` 输入 `Hummingbird MD`
